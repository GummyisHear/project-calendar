Microsoft Project

A project management tool by Microsoft — explained simply.

What is it?

Microsoft Project is software that helps project managers plan work, assign resources, track progress, and manage budgets. First released in 1984, it is one of the most widely used PM tools in construction, IT, engineering, and government.

Core features

  • Gantt charts — visualise tasks as bars on a timeline with dependencies.
  • Task dependencies — link tasks so date changes cascade automatically.
  • Resource management — assign people, track costs, and resolve over-allocations.
  • Critical path — highlights the task chain that controls the project end date.
  • Baselines — save a snapshot and compare planned vs. actual progress.
  • Microsoft 365 integration — connects with Teams, SharePoint, and Power BI.
